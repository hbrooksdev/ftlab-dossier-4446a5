04

Key concepts

The two jobs of hierarchy
Hierarchy did two separable things: route information (which AI now dissolves) and locate accountability (which it does not). The whole design turns on this split.
Mirroring hypothesis
An organisation’s structure mirrors the architecture of the system it builds (Conway’s Law). For a studio that builds software with software agents, the org and the system are one object — so design the architecture and let the org follow.
Machine Core + Human Cortex
FTLAB’s own primitive: autonomous systems carry execution; a small human team carries intent, judgment, and values. It is the two-jobs split, stated in the framework.
The New Triumvirate
Distributed governance in three functions — Architect (how), Intent-Setter (why), Guardian (what) — held in deliberate tension, never concentrated in one executive.
The Core
Option 2’s coordination substrate: an artifact layer → a world model (engineered as a database) → a capability registry → routing + verification agents. It carries the routing job middle management used to.
Engines
The value-producing work as head-agents orchestrating agent swarms (the Knowledge Flywheel), coordinating through the Core rather than a manager.
The Edge
The small, flat human team on the boundary — policy, exception-handling, verification, and the legal/accountability floor. The job AI does not dissolve.
Topology-by-task
Match the coordination pattern to the work: parallelise reads/research, single-thread writes/synthesis. Architecture-task alignment, not agent count, drives success.
Composition-failure → venture thesis
When the system can’t compose a solution to a real need from existing capabilities, that gap is logged as a candidate venture — coordination doubles as deal origination.
Fractal instantiation
Every venture is its own Core/Engines/Edge instance, coordinated through a thin shared “waist.” Governance scales by repeating a self-similar pattern, not by adding management layers.
The permanent floor
“Fully autonomous” can be true of doing and coordinating, never of accountability — legitimacy, values authorship, and the final halt stay human, by law, not by capability.
The four steal-ables
From Option 4, improving any design now: price human attention, make reputation first-class (autonomy earned), compile Values into rules-as-code, add a pressure/selection signal to the substrate.
